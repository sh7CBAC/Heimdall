# Ourenus subscription template preset

This directory contains the Heimdall-bundled Ourenus subscription page preset.

Original project:
- https://github.com/MatinDehghanian/Ourenus

Runtime target path used by Heimdall:
- /usr/local/x-ui/sub_templates/ourenus/index.html

Heimdall serves this template through the existing `subThemeDir` setting and the subscription page renderer.
The backend compatibility endpoint for this template is:

- GET /sub/:subid/info
